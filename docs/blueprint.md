# **App Name**: MotoTrust Locator

## Core Features:

- Dealership Locator: Display a map with motorcycle dealership locations based on OpenStreetMap data.
- Dealership Filtering: Allow users to filter dealerships by brand and service type.
- Dealership Details: Show detailed information about each dealership, including name, brand, address, phone number, stock count, and rating.
- Mobile-Responsive Layout: Adapt the layout to both desktop (sidebar + map) and mobile (full-screen map with sliding drawer) screen sizes.
- Ad Space Integration: Include designated ad spaces for banners within the app layout to display banner ads and other relevant information.
- Motorcycle Advice Articles: Display articles for motorcycle advice and information.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to evoke trust and reliability, inspired by classic motorcycle brands.
- Background color: Very light blue-gray (#F0F4F8), providing a neutral backdrop that is easy on the eyes.
- Accent color: A muted Purple (#9FA8DA), analogous to the primary blue, providing visual interest without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif font, for clean and modern readability on both desktop and mobile devices.
- Use 'lucide-react' icons for a consistent and modern visual language. Icons should be simple, clear, and related to the app's functions.
- Desktop: Left sidebar for list, right side for map. Mobile: Map fills the screen, list is a retractable drawer from the bottom.
- Subtle animations on map marker selection and drawer transitions to improve UX.
